# Claude Code 使用教程

## 简介

Claude Code 是 **Anthropic 公司**做的一款 **AI 编程助手**,类似"会写代码的 ChatGPT"。

你可以把它当作一个"超级程序员搭档",它可以帮你:

- **写代码**:比如写 Python、JavaScript、Go 等语言。
- **改代码**:你写了一半,它帮你优化、补全。
- **解释代码**:看不懂报错?它能解释问题出在哪。
- **生成文档**:自动生成注释、接口说明、开发文档。
- **做计划**:你给一个目标,它会拆分成开发步骤。

**简单说**:你不用自己从零到一写完,Claude Code 可以帮你分担 60%~80% 的工作。

## 一、Claude Code 安装

### 一)前提条件

#### 1、NodeJS

如果已经安装过就不用再安装

##### 1.1 windows

官网地址:https://nodejs.org/

直接下载并选择安装程序

##### 1.2、mac

访问 https://nodejs.org/ 下载 macOS 版本并安装

#### 2、Git

##### 2.1、windows

下载 Git for Windows 并安装

##### 2.2、mac

通过 Homebrew 或官网下载安装

#### 3、安装 claude code

在终端中运行安装命令安装 Claude Code CLI

## 三、Claude code 启动

### 一)启动终端

#### 1、windows

##### 1.1、方式一:右键打开终端

在项目文件夹中右键选择"在终端中打开"

##### 1.2、方式二:资源管理器的地址栏

在资源管理器地址栏输入 `cmd` 或 `powershell`

#### 2、mac

使用 Spotlight 搜索"终端"或在文件夹中右键选择"新建位于文件夹位置的终端窗口"

#### 3、运行 claude

在终端中输入:
```bash
claude
```

## 四、开始使用 Claude Code

### 案例:贪吃蛇游戏

#### 步骤一:新建目录

创建一个新的项目目录

#### 步骤二:进入目录,使用终端打开

在该目录中打开终端

#### 步骤三:终端输入 claude

运行 `claude` 命令启动

#### 步骤四:输入提示词

输入你的需求,例如:"请帮我创建一个贪吃蛇游戏"

#### 步骤五:打开游戏

在浏览器中打开生成的 HTML 文件

#### 步骤六:调试问题

如果有问题,向 Claude Code 描述问题并让它修复

## 五、Claude code 常用技巧

### 一)常用命令

- `/help` - 显示帮助信息
- `/clear` - 清除当前对话
- `/new` - 开始新对话
- `/history` - 查看历史记录

### 二)常用方式

#### 1. 指定文件

明确告诉 Claude Code 要修改哪个文件

#### 2. 粘贴图片

可以直接粘贴设计稿或界面截图

#### 3. 深度思考

让 AI 更深入地分析问题

#### 4. 自动编辑模式

AI 自动编辑文件而不需要手动确认

#### 5. plan 模式

让 AI 先制定计划再执行

### 三)常用slash commands

使用 `/` 开头的命令来执行特定操作

### 四)yolo 模式(狂飙模式)

启用快速开发模式,AI 会更激进地修改代码

### 五)记忆文件

#### 1、全局记忆文件

在 `~/.claude/` 目录下创建 `CLAUDE.md` 文件,设置全局规则

#### 2、项目记忆文件

在项目根目录创建 `CLAUDE.md` 文件,设置项目特定规则

### 六)恢复对话/查看聊天记录

使用 `/history` 命令查看和恢复之前的对话

### 七)监控使用量

查看 API 调用次数和 token 使用情况

## 最佳实践建议

1. **清晰的需求描述**:越详细越好
2. **分步骤执行**:不要一次性提出太多要求
3. **及时反馈**:遇到问题立即告知 Claude Code
4. **使用 CLAUDE.md**:建立项目规范和偏好设置
5. **保持耐心**:AI 也需要迭代和优化

---

## 注意事项

- 确保网络连接稳定
- API Key 需要有足够的配额
- 重要代码记得备份
- 理解生成的代码再使用
- 保护好你的 API Key

## 总结

Claude Code 是一个强大的 AI 编程助手,能够显著提高开发效率。通过合理使用,你可以:

- 快速原型开发
- 学习新技术和框架
- 解决复杂的编程问题
- 生成文档和测试代码
- 重构和优化现有代码

记住:**AI 是助手,不是替代品**。最终的代码质量和架构决策仍然需要你来把关。
